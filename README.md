# Maharashtra Government Schools — Data Dashboard

An interactive web dashboard for exploring multi-year government school data across all 36 districts of Maharashtra, India. Data is sourced from the **UDISE+ Microdata Portal** and served live from a local SQLite database.

---

## Live Demo

Start the server:

```bash
python server.py
```

Then open **http://localhost:8080** in your browser.

## Dashboard Features

- **Summary statistics** — total schools, primary, upper primary, secondary, higher secondary (latest year)
- **Year-wise trend chart** — total government schools from 2018-19 to 2024-25 with AI forecast
- **Category breakdown** — Primary / Upper Primary / Secondary / Higher Secondary (doughnut chart)
- **Management type distribution** — Government, Local Body, Aided, Private Unaided (bar chart)
- **Year-on-Year change (%)** — bar chart showing growth/decline per year
- **AI Forecast** — 3-year linear regression prediction with R² accuracy metric
- **Year-wise summary table** — full breakdown across all years
- **District-wise table** — all 36 districts with search and sort controls

## Data Source

| Source | URL |
|--------|-----|
| **UDISE+ Microdata** | [microdata.udiseplus.gov.in/getCsvData](https://microdata.udiseplus.gov.in/getCsvData) |

Data covers **Maharashtra State** — Government, Local Body, and Government-Aided schools — from **2018-19 to 2024-25** across all **36 districts**.

The raw CSV files downloaded from UDISE+ are loaded into a local SQLite database (`resources/schools.db`) containing **22.4 million rows** across 6 tables:

| Table | Description | Rows |
|-------|-------------|------|
| `school_profile_1` | School identity, location, type, medium | 764,828 |
| `school_profile_2` | Schemes, inspections, SMC, grants | 764,828 |
| `school_enrolment_1` | Enrolment by class & gender (groups 1–7) | 8,160,798 |
| `school_enrolment_2` | Enrolment by class & gender (groups 8+) | 11,200,101 |
| `school_facility` | Infrastructure, toilets, labs, ICT | 764,828 |
| `school_teacher` | Teacher counts by gender, category, qualification | 764,828 |

## Setup

### Prerequisites

- Python 3.9+
- `numpy` (for AI forecasting)

### Install dependencies

```bash
cd scraper
pip install -r requirements.txt
```

### Build the database (first-time setup)

> **Note:** `resources/schools.db` (2.3 GB) is not included in the repository — it exceeds GitHub's file size limits. Build it locally using the steps below.

Download the UDISE+ CSV zip files for Maharashtra from [microdata.udiseplus.gov.in/getCsvData](https://microdata.udiseplus.gov.in/getCsvData) and place them in the `resources/` folder, then run:

```bash
cd scraper
python load_to_db.py
```

### Run the dashboard

```bash
python server.py
```

Open **http://localhost:8080**.
