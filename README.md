# Maharashtra Government Schools — Data Dashboard

An interactive web dashboard for exploring multi-year government school data across all 36 districts of Maharashtra, India. Includes a Python scraper that fetches live data from public government portals and generates the JSON dataset powering the dashboard.

---

## Live Demo

Serve locally with:

```bash
npx serve web
```

Then open **http://localhost:3000** in your browser.

## Dashboard Features

- **Summary statistics** — total schools, primary, secondary, higher secondary (latest year)
- **Year-wise trend chart** — total government schools from 2012-13 to 2023-24
- **Category breakdown** — Primary / Upper Primary / Secondary / Higher Secondary (pie chart)
- **Management type distribution** — Government, Local Body, Aided, Private Unaided
- **Year-on-Year change (%)** — bar chart showing growth/decline per year
- **Year-wise summary table** — full breakdown across all years
- **District-wise table** — all 36 districts with search and sort controls

## Data Sources

| Source | Description |
|--------|-------------|
| [UDISE+ Portal](https://udiseplus.gov.in) | Unified District Information System for Education — primary source |
| [data.gov.in](https://data.gov.in) | Government of India Open Data Portal |
| [Ministry of Education](https://www.education.gov.in) | Annual reports and school statistics |
| [Census of India](https://censusindia.gov.in) | Educational infrastructure data |

Data covers **Maharashtra State** — Government, Local Body, and Government-Aided schools — from **2012-13 to 2023-24** across all **36 districts**.

## Scraper Setup

### Prerequisites
- Python 3.9+

### Install dependencies

```bash
cd scraper
pip install -r requirements.txt
```

### Run the scraper

```bash
python scrape.py
```

This fetches live data from all four government portals and writes the output to `web/data/schools_data.json`. If a live fetch fails, it falls back to compiled UDISE+ report data.
